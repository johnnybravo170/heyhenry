Brief alignment

  • Action-first ordering — orientation line → celebration banner → needs-you → chase list → audit (activity). The dashboard is for triage and action; standing-number tiles live on their own screens (Business Health / Billing / Projects), one tap away in the nav.
  • Henry as a one-line orientation — the multi-line digest is gone. Right under the greeting, Henry contributes a single count + read of the day: “3 things need you today” (rust mark, working day), or “Nothing needs you — you’re caught up” (green check, clear day). It orients, never enumerates — the items live in the cards below so nothing repeats.
  • Each signal lives in exactly one place — wins in the green celebration banner, money / follow-ups / aging-estimates in the chase list, day-to-day tasks in Today / Blocked / Needs-You. Henry is the read + embedded flags, not a digest that repeats the cards.
  • Needs You — the action zone as 3 sub-cards: Today (overdue + due, 4 items), Blocked (5, tagged Client/Material/Sub so the owner knows who to chase), Needs You (2 decisions with inline Verify/Approve buttons).
  • One chase list, money first — Needs Attention + Money-at-risk merged into a single ranked list of 4: Patel overdue invoice (with the “Henry chased · no reply” pill + “I called them” dismiss), Dawson stale quote, Lin Family aging estimate ($18,150, sent 5d, not viewed — “Nudge”), Sarah Chen follow-up. Each row deep-links to its source surface. Money appears only here.
  • Recent Activity demoted — 5 latest only, behind a “View all” expander on default; expanded variant shows the full audit feed. Never dominates the screen.
  • What was removed — TodaysJobs, getJobTaskHealth, generic PipelineSummary, every “coming soon” placeholder, AND the de-duped 6-tile metric strip itself (Planning / Awaiting / Active / Complete / Revenue / Outstanding). Those numbers live on Business Health, Billing, and Projects — the dashboard is for triage, not for re-displaying metrics. GC-only path; no isRenovation branching anywhere.
Notes: the celebration banner is dismissible — once Jonathan opens or dismisses the Graham-accepted message it disappears from this card. Money appears only in the chase list, with the Patel overdue row in danger red because it is the only red-priority money row; the Lin estimate row carries its $18,150 in plain ink because aging-but-not-yet-late doesn’t earn alarm color. Three type sizes per screen (28 / 16 / 14 / 12 mixed across heading-scale and body-scale, but body uses only 16 / 14 / 12).
